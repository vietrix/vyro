# Installation

## Prerequis

- Python 3.10+
- pip
- Rust toolchain (uniquement pour build depuis les sources)

## Installation depuis PyPI

```bash
pip install vyro
```

## Verification

Commande principale (recommandee):

```bash
vyro --help
```

Fallback si PATH n'est pas pret:

```bash
python -m vyro --help
```
